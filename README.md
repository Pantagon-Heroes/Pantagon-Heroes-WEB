# Pantagon Heroes Web 0.0.0

This is a landing page website for the mobile game pantagon heroes, actually made in react, and react-native.

### Prerequisites


```
React
Node
```

### Installing
you need to have npm installed on your machine
```
npm install
```

And repeat

```
npm start
```
